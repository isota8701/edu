# 수학 — 세 자리 수 어림·비교 심화 (1단원 · 2세션)

> 1세션은 모두 정답이었어요. 오늘은 같은 단원에서 한 칸만 더 깊이 들어갑니다. 새로운 문제 유형이 살짝 섞여 있어요. **막히는 게 정상**이라는 점, 시작 전에 아이에게 한 번 짚어주세요.
> 예상 시간: **약 40~45분**

## 1. 워밍업 — 구구단 4단 (3~4분)

부모가 식을 불러주고, 아이가 곧바로 답하면 ✓, 잠깐 멈칫하면 △ 표시.
순서는 **섞어서** 불러주세요 (표 위에서 아래로 가지 말고).

| 식 | 답 | 응답 |
|----|----|------|
| 4 × 1 | | ✓ / △ |
| 4 × 7 | | ✓ / △ |
| 4 × 4 | | ✓ / △ |
| 4 × 5 | | ✓ / △ |
| 4 × 6 | | ✓ / △ |
| 4 × 3 | | ✓ / △ |

> **△가 3개 이상이면**: 워밍업 끝나고 △ 표시한 식만 골라 한 번 더 천천히 불러주세요. 답을 맞히는 게 목적이 아니라 **같은 식을 두 번 떠올리는 경험**이 자동화로 이어집니다.
> **△가 2개 이하면**: 잘 가고 있어요. 다음 세션부터 4×6, 4×8을 한 번씩 더 끼워 넣을 거예요.

## 2. 개념 확인 — 수 어림·비교 (5분)

**핵심 메시지 (아이에게 그대로 읽어주세요)**
> "어림은 정확한 답이 아니야. **'대략 어디쯤'**을 보는 눈이야. 책 읽다가 '한 200쪽쯤 되겠다' 할 때 그게 어림이야."

**부모가 묻는 질문**
1. "187은 100에 더 가까울까, 200에 더 가까울까? 왜 그렇게 생각해?"
2. "423하고 478, 둘 다 400대인데 어느 쪽이 500에 더 가까워?"
3. "수직선에 350을 표시하면, 300하고 400 중에 어느 쪽에 더 가까이 그릴까?"

> 정답을 채점하지 말고, **"왜?"** 한 번 더 묻고 넘어가세요. 이 단계는 본문제 워밍업입니다.

## 3. 본문제 (20~25분, 7문제)

> 부모가 연습장에 옮겨 적어주거나 천천히 읽어주세요.

**(1) 자릿값 응용**
> 백의 자리 수와 일의 자리 수를 더하면 십의 자리 수가 되는 세 자리 수를 **하나** 만들어보자.
> (예: 백의 자리가 2, 일의 자리가 3이면 십의 자리는 ?)

**(2) 카드 수 만들기 — 두 번째로 큰 수**
> 숫자 카드 **3, 7, 5, 1** 중에서 세 장을 골라 세 자리 수를 만들 거야.
> ① 만들 수 있는 **가장 큰** 수는?
> ② 만들 수 있는 **두 번째로 큰** 수는?
> ③ 두 수의 차는 얼마야?

**(3) 수직선 응용 — 두 수 사이 거리**

```
0 ---|---|---|---|---|---|---|---|---|---|--- 1000
     100  200  300  400  500  600  700  800  900
```

> 수직선 위에 **270**과 **640**을 손가락으로 짚어보자.
> ① 270은 200과 300 중 어디에 더 가까워?
> ② 640은 600과 700 중 어디에 더 가까워?
> ③ 두 수 사이 거리는 대략 얼마쯤 될까? (정확한 계산 X, 어림으로)

**(4) 어림 — 어느 쪽이 더 가까울까**
> 다음 수들이 **몇백에 가장 가까운지** 말해보자.
>
> ① 187 → ( )
> ② 350 → ( )  ← 살짝 함정
> ③ 612 → ( )
> ④ 749 → ( )

**(5) 크기 비교 — □ 채우기**
> □ 안에 들어갈 수 있는 숫자를 **모두** 찾아보자.
>
> ① 4□6 < 460 → □ =
> ② 5□8 > 528 → □ =
> ③ 3□0 > 380 → □ = (조심: '같다'는 안 돼)

**(6) 짧은 문장제**
> 도서관에 책이 **368권** 있어. 어제 새 책 **45권**이 들어왔고, 오늘 누군가 **22권**을 빌려갔어.
> ① 지금 도서관에 있는 책은 몇 권이야?
> ② 그 수는 **400에 더 가까울까, 300에 더 가까울까?**

**(7) 거꾸로 풀기 (가벼운 도입)**
> 어떤 세 자리 수가 있어. 그 수에서 **백의 자리만 1 줄였더니 528**이 되었어.
> 처음 수는 무엇이었을까? 어떻게 알았는지도 말해보자.

## 4. 사고력 도전 ★ (10분, 2문제)

> "이건 어른도 잠깐 멈춰. 끝까지 못 풀어도 괜찮아. **시도한 게 답이야.**"

### ★ 1 — 자릿수 조건 퍼즐
> **자릿수의 합이 12**이고, **십의 자리가 0**인 세 자리 수를 모두 찾아보자.
> (예: 백의 자리 + 0 + 일의 자리 = 12)

- **힌트 1**: "백의 자리에 1을 넣어보자. 그러면 일의 자리는 얼마가 되어야 할까?"
- **힌트 2**: "백의 자리를 1, 2, 3 … 9까지 차례로 넣어보면서 적어볼래?"

### ★ 2 — 뛰어 세기 + 어림 융합
> **1, 4, 7, 10, 13, …** 이렇게 **3씩 뛰는 수**가 있어.
> 이 수들 중에서 **100과 가장 가까운 수**는 무엇일까?

- **힌트 1**: "100 근처까지 가야 해. 30번 정도 뛰면 어디쯤일지 어림해보자. (3 × 30 = ?)"
- **힌트 2**: "100을 살짝 넘는 수, 살짝 안 되는 수 두 개 찾아서 비교해봐."

## 5. 부모용 정답 + 풀이 포인트

| 번호 | 답 | 핵심 포인트 / 흔한 실수 |
|------|----|------|
| 1 | 예: **235**, **123**, **347**, **549** 등 (백+일=십이면 OK) | 답이 여러 개라는 것 자체가 학습 포인트. "또 다른 거 있어?" 한 번 더 물어보기. 백+일이 10 이상이면 안 됨 (예: 5_8은 십의 자리에 13을 못 넣음) |
| 2 | ① 753  ② 751  ③ 753 - 751 = **2** | "두 번째로 큰"을 만들 때 일의 자리만 한 칸 내리는 감각. 7-5-3 다음은 7-5-1이라는 점. 차가 2라는 게 직관에 어긋날 수 있어요 — "왜 작아 보이는데 차가 2뿐일까?" 짚어주기 |
| 3 | ① **300에 가까움** (200까지 70, 300까지 30)  ② **600에 가까움** (600까지 40, 700까지 60)  ③ 약 **370 정도** (정답 범위 350~400 OK) | ③번은 정확한 뺄셈 안 해도 됨. "수직선에서 손가락으로 거리 가늠"이 목적. 640 - 270 = 370 정확히 맞히면 보너스 |
| 4 | ① 200  ② 300 또는 400 (둘 다 정답)  ③ 600  ④ 700 | ②번 350은 **딱 중간**. 어느 쪽으로 답해도 정답. "왜 헷갈려?"라고 물어 중간값 개념을 자연스럽게 끌어내기 |
| 5 | ① **0, 1, 2, 3, 4, 5** (6은 안 됨, 같아짐)  ② **3, 4, 5, 6, 7, 8, 9** (2도 528=528이라 X)  ③ **9** (380보다 커야 하니 39_는 9만, 다른 자리는 안 됨) | ⑤-③이 가장 까다로움. "3□0이 380보다 크려면 십의 자리가 8보다 커야 한다"는 자릿값 사고. □=8이면 380=380이라 부등호 X |
| 6 | ① 368 + 45 - 22 = **391권**  ② **400에 더 가까움** (400까지 9, 300까지 91) | 세로셈 두 번. 받아올림 한 번 있음 (368+45). ②는 어림 적용 — 세 자리 수 끝자리만 봐도 답이 보임 |
| 7 | **628** | "백의 자리만 1 줄여서 528 → 원래는 628". 거꾸로 풀기 입문. "그러면 십·일의 자리는 안 변했지?" 짚어주기 |
| ★1 | **309, 408, 507, 606, 705, 804, 903** (7개) | 풀이 경로: 백의 자리를 1부터 9까지 대입. 1+0+11(X), 2+0+10(X), 3+0+9(O) … 처음 두 개는 일의 자리가 10 넘어서 탈락하는 게 핵심 발견. 5개 이상 찾으면 충분. 7개 다 찾으면 박수 |
| ★2 | **100** (1, 4, 7, …, 97, 100 — 100 자체가 수열에 포함) | 풀이: 1에서 시작해 3씩 더하면 1, 4, 7, 10, 13, … 일반화하면 "3으로 나눈 나머지가 1인 수". 100 ÷ 3 = 33 나머지 1 → **100이 정확히 이 수열에 들어감**. "가장 가까운"이 아니라 100 자체가 답. 만약 아이가 97과 100을 다 찾고 "100이 더 가까워, 거리 0이야"라고 말하면 완벽 |

## 6. 흔한 실수 패턴

- **5번 □ 채우기**: "**모두** 찾기"를 놓치고 한두 개만 쓰고 넘어가는 경향. "또 있어?" 한 번 더 묻기.
- **4번 350 같은 중간값**: 한쪽으로 강제로 정하려고 함. 중간값은 둘 다 정답이라고 알려주기.
- **★2 뛰어 세기**: 100 근처에서 손으로 일일이 더하다 지침. 곱셈(3×33=99, +1=100)으로 점프하는 감각을 살짝 보여주기.
- **거꾸로 풀기 (7번)**: "1 줄였다"를 "1 더했다"로 거꾸로 처리해야 하는데, 그대로 528에서 1을 또 빼는 경우. "지금 답이 528이지? 원래는 더 컸어, 작았어?"로 방향 잡아주기.
