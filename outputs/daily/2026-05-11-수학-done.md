## 1. 워밍업 (3~5분)

### 1-1. 47+19 다시 보기 — 19를 갈라서 50 만들기

> 부모 멘트: "47에 몇을 더하면 딱 50이 될까?"

- 47 + 3 = 50
- 19는 3과 □로 가를 수 있다.
- 그래서 47+19 = 47+3+□ = 50+□ = □

정답: 19=3+16, 47+19=66

> 지도 포인트: 오늘 핵심은 47+19를 맞히는 것이 아니라, **19를 3과 16으로 갈라 쓴 이유**를 말하게 하는 것.

### 1-2. 중간값 이름 붙이기

```
42 - 17 + 37
```

먼저 계산할 부분:
- 42-17 = □
- □+37 = □

정답: 25, 62

> 부모 멘트: "첫 번째 □는 끝난 답이 아니라, 다음 계산으로 가져갈 중간값이야."

### 1-3. 5씩 묶기 한 번

사탕이 5개씩 6봉지 있습니다. 모두 몇 개인가요?

- 식: 5×□=□

정답: 5×6=30

## 3. 본문제 (20~25분)

### (1) 시험 유형 복원 — 십 만들기 과정식 [누적 — 과정식]

□ 안에 알맞은 수를 써 보세요.

```
47+19 = 47+3+□ = 50+□ = □
```

> 부모 멘트: "19를 아무렇게나 가르는 게 아니라, 47을 50으로 만들 만큼 먼저 떼어 주는 거야."

---

### (2) 새 수로 해 보기 — 58+26 [누적 — 과정식]

58을 60으로 만들려고 합니다. □ 안에 알맞은 수를 써 보세요.

```
58+26 = 58+2+□ = 60+□ = □
```

질문: 26을 2와 몇으로 갈랐나요?

---

### (3) 한 번 더 — 36+28 [누적 — 과정식]

36을 40으로 만들려고 합니다.

```
36+28 = 36+4+□ = 40+□ = □
```

질문: 왜 28에서 4를 먼저 떼었나요?

---

### (4) 순서도 칸 채우기 — 42-17+37 [누적 — 중간값 기록]

아래 식의 값을 구하려고 합니다. □ 안에 알맞은 수를 쓰세요.

```
42-17+37 = □
```

순서대로:

```
42-17 = □
□+37 = □
```

> 부모 멘트: "위 □와 아래 첫 번째 □가 같은 수인지 확인해 보자. 중간값을 다음 줄로 옮기는 게 포인트야."

---

### (5) 곱셈 문장제도 중간값으로 풀기 [새 진도 — 곱셈+덧셈]

스티커가 5개씩 6줄 있고, 낱개 스티커가 17개 더 있습니다. 스티커는 모두 몇 개인가요?

순서대로 써 보세요.

```
5×6 = □
□+17 = □
```

답: □개

---

### (6) 가게 문제 — 팔고 더 들어온 인형 [새 진도 — 순서도]

가게에 곰 인형이 24개 있었습니다. 오전에 8개를 팔고, 오후에 15개가 새로 들어왔습니다. 지금 곰 인형은 몇 개인가요?

순서대로:

```
24-8 = □
□+15 = □
```

식 한 줄로도 써 보세요.

```
24-8+15 = □
```

---

### (7) 5단과 받아올림 연결 [새 진도 + 누적]

초콜릿이 5개씩 7상자 있고, 사탕이 28개 있습니다. 초콜릿과 사탕은 모두 몇 개인가요?

순서대로:

```
5×7 = □
□+28 = □
```

> 부모 멘트: "곱셈으로 먼저 묶음의 수를 구하고, 그 다음 덧셈으로 모두 합치는 문제야."

---

### (8) 4단·5단 비교 [기초 점검 — 자연 노출]

다음 두 묶음 중 어느 쪽이 더 많고, 몇 개 더 많은가요?

- A: 쿠키 5개씩 7봉지
- B: 쿠키 4개씩 8봉지

순서대로:

```
A: 5×7 = □
B: 4×8 = □
차이: □-□ = □
```

## 4. 사고력 도전 (8~10분)

> 이건 어른도 잠깐 멈춰. 끝까지 못 풀어도 괜찮아. 어디까지 생각했는지가 더 중요해.

### 도전 1 — 십 만들기 전략 [사고 방향: 수 분해]

다음 식을 "가까운 십"을 먼저 만들도록 바꾸어 써 보세요.

```
69+18 = 69+□+□ = 70+□ = □
```

질문:
- 69를 70으로 만들려면 몇이 필요할까요?
- 18은 그 수와 나머지 몇으로 갈라지나요?

---

### 도전 2 — 답이 24가 되는 곱셈식 모두 찾기 [사고 방향: 체계적 탐색]

답이 24가 되는 곱셈식을 모두 찾아 보세요. 단, 1×24와 24×1도 빠뜨리지 말고 적어 봅니다.

힌트 표:

| 앞의 수 | 24가 되려면 뒤의 수는? | 곱셈식 |
|------|------------------|--------|
| 1 | □ | 1×□=24 |
| 2 | □ | 2×□=24 |
| 3 | □ | 3×□=24 |
| 4 | □ | 4×□=24 |
| 5 | 안 됨 | — |
| 6 | □ | 6×□=24 |

> 부모 멘트: "지난번엔 20에서 10×2를 놓쳤지. 오늘은 작은 수부터 차례로 표에 적어 보자."
