## 오늘의 지문: 「작은 새와 큰 나무」 · 문학(창작 우화)

[지문 본문]

숲속 작은 새 한 마리가 큰 나무 위에 둥지를 지었어요.

어느 날, 거센 바람이 불어와 둥지가 흔들렸어요. 작은 새는 무서워서 나무에게 말했어요.
"나무야, 네가 너무 흔들려서 둥지가 떨어질 것 같아. 좀 가만히 있어 줘."

나무가 가지를 살짝 굽히며 대답했어요.
"미안해, 작은 새야. 하지만 내가 바람에 흔들리지 않으면, 나는 부러지고 말 거야. 흔들리는 건, 부러지지 않으려고 그러는 거란다."

작은 새는 가만히 생각했어요. 다음 날, 새는 둥지를 더 깊은 가지 사이에 단단히 다시 지었어요. 그리고 나무에게 말했어요.
"이제 네가 흔들려도 괜찮아. 나도 흔들리는 법을 배웠거든."

## 읽고 답하기

### 사실 확인
1. 작은 새는 처음에 왜 무서워했나요?
2. 나무는 왜 바람이 불 때 흔들린다고 했나요?

### 어휘
3. 글에 나온 "거센"은 어떤 느낌일까? (보기 중에 골라도 좋고, 너의 말로 말해도 좋아)
   - ㄱ. 아주 약하고 부드러운
   - ㄴ. 아주 세고 거친
   - ㄷ. 따뜻하고 포근한

### 추론
4. 작은 새는 왜 둥지를 더 깊은 가지 사이에 다시 지었을까?
5. "나도 흔들리는 법을 배웠거든"이라고 새가 말한 건 무슨 뜻일까? (글자 그대로 흔들리는 게 아니라, 마음으로는 어떤 뜻일까?)

### 의견
6. 너는 작은 새처럼 무서운 일이 있을 때, 어떻게 하는 편이야? 한두 문장으로 말해줘.

## 토론 질문 (부모와 함께, 1~2분)

> "흔들리는 건 나쁜 걸까, 좋은 걸까?"
> 너무 흔들리지 않으려고 단단해지면 오히려 부러질 수도 있고, 너무 흔들리면 둥지가 떨어질 수도 있어. 어떻게 하면 좋을까?

### 부모용 토론 가이드
- **예시 답변 A (흔들림은 필요해 입장)**: "흔들려야 부러지지 않아. 사람도 힘들 때 울거나 쉬어야 다시 일어날 수 있어." → 명언 연결: "휘어지는 나무가 부러지지 않는다."
- **예시 답변 B (단단함도 필요해 입장)**: "그래도 너무 흔들리면 위에 있는 둥지가 떨어지니까, 중요한 건 잘 지켜야 해." → 책임감·중심을 잡는 일에 대해 이야기.
- **아이가 짧게 답하면 던질 후속 질문**:
  - "너는 학교에서 친구랑 다툰 날 같은 거, 흔들린 적 있었어?"
  - "그때 너는 나무처럼 흔들렸을까, 작은 새처럼 둥지를 옮겼을까?"
  - "둘 다 필요한 게 아닐까? 어떨 때 흔들리고, 어떨 때 단단해야 할까?"
- **부모용 메모**: 결론을 내릴 필요 없음. "흔들림이 꼭 나쁜 게 아니다"라는 감각이 한 번 스치면 충분.

## 부모용 정답·해설

| 번호 | 답 | 해설 |
|---|---|---|
| 1 | 거센 바람에 나무가 흔들려서 둥지가 떨어질 것 같았기 때문 | **사실 확인**: 글 속에서 직접 답을 찾는 문제. "왜"라는 말 때문에 추론처럼 보일 수 있지만, 본문에 거의 그대로 적혀 있음. |
| 2 | 흔들리지 않으면 부러지기 때문 | **사실 확인**: 본문 인용 가능. 아이가 자기 말로 풀어 말하면 더 좋음. |
| 3 | ㄴ. 아주 세고 거친 | **어휘 추론**: "거센 바람"은 초2 수준에서 약간 도전적. 문맥(둥지가 흔들림 → 약한 바람 아님)에서 추론 가능. 아이가 "엄청 센 바람"처럼 자기 말로 표현하면 정답. |
| 4 | 바람에 흔들려도 떨어지지 않게 하려고 / 둥지를 더 안전하게 만들려고 | **추론**: 글에 직접 답이 없음. 새가 나무의 말을 듣고 행동을 바꿨다는 인과를 잡아야 함. |
| 5 | 나무처럼 무서운 일이 있어도 받아들이는 법 / 흔들림(어려움)을 두려워하지 않는 마음을 갖게 됐다는 뜻 | **추론(상징)**: 가장 어려운 문제. 비유를 잡지 못하면 "글자 그대로" 답할 수 있음. 그래도 OK — 비유 이해는 초2 후반에 본격 진입. 부모가 살짝 풀어 설명해 주는 것도 좋음. |
| 6 | 자유 답 | **의견**: 정답 없음. 아이의 평소 모습이 드러남. "도망간다", "엄마한테 말한다", "참는다" 모두 OK. |

### 5단계별 진단 포인트 (부모가 관찰)
- **1·2번 (사실 확인)**: 본문을 다시 들춰 보며 답 찾는지, 기억으로 답하는지 → 둘 다 OK. 본문 다시 보는 건 좋은 습관.
- **3번 (어휘)**: 모르면 "거센 바람" 한 번 더 천천히 읽고, 어떤 바람일지 그림 그려보게 유도.
- **4번 (추론)**: 답을 못 찾으면 "새가 나무 말을 듣고 뭔가 바꿨지? 뭘 바꿨어?" 단서 주기.
- **5번 (추론·상징)**: 어려우면 패스해도 됨. **"이 단계가 어려운지"가 진단 포인트**.
- **6번 (의견)**: 길게 답하는지 / 짧게 끝내는지 / 자기 경험 꺼내는지 관찰.
