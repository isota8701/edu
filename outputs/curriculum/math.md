## 2. 곱셈의 의미 + 4·5단 정착

- **기간**: 3주 (6~7세션)
- **핵심 개념**: 곱셈 = 같은 수 반복 더하기 / 묶어 세기 / 직사각형 배열 모델 / 교환법칙(4×3 = 3×4)
- **학교 진도 sync**: 1학기 곱셈 도입과 동기화
- **시작 진단** (1세션 첫 5분): "사과 4개씩 3묶음 → 모두 몇 개?" + 4단 5문제 + 5단 3문제 (5단 노출 후 첫 반응 보기)
- **종료 평가**: 8문제 (곱셈 모델 추론 3, 곱셈식↔덧셈식 변환 2, 4·5단 자동화 3 — 통과 6/8)
- **세션 패턴**:
  - 워밍업: 5단 신규 노출 + 4단 누적 랜덤 (2~3분)
  - 본문제: 묶음 그림→곱셈식, 곱셈식↔덧셈식, 직사각형 배열에서 식 만들기, 5단 활용 묶음 응용
  - 도전: "4×3과 3×4가 왜 같을까? 그림으로 설명" / "5×4 ≠ 5+4를 그림으로 보여주기"

### 5단 도입 페이스
- 1세션: 5단 = 5씩 뛰기와 같음을 보여주기 (5, 10, 15, 20…). 5×1 ~ 5×5만.
- 2세션: 5×6 ~ 5×9 추가. 5의 배수 패턴(끝자리 0 또는 5).
- 3세션: 5단 랜덤 응답 시도. 막힘 1초 이상이면 △.

## 3. 두 자리 덧뺄셈 사고력 응용

- **기간**: 2주 (4~5세션)
- **핵심 개념**: 받아올림/내림 안정화 + 가르기·모으기 전략 다양화 + 거꾸로 풀기 + 빈칸 채우기 응용
- **학교 진도 sync**: 1학기 덧뺄셈 단원과 동기화 (이미 학교에서 진행 중)
- **시작 진단**: 받아내림 1문제 + 받아올림 1문제 (속도 측정)
- **종료 평가**: 5문제 (받아올림 1, 받아내림 2, 가르기/모으기 응용 1, 거꾸로 풀기 1)
- **세션 패턴**:
  - 워밍업: 5단 자동화 + 6단 도입(맛보기)
  - 본문제: 빈칸 □ 채우기 (`34 + □ = 71`), 거꾸로 풀기, 한 문제 두 가지 풀이
  - 도전: 짧은 문장제 (예: 동생에게 몇 개 줬더니 → 거꾸로)

## 4. 길이 재기 (cm·m)

- **기간**: 1.5주 (3~4세션)
- **핵심 개념**: cm·m 단위 의미 / 자로 재는 법 / 길이 어림 / 단위 환산(1m = 100cm)
- **학교 진도 sync**: 1학기 길이 단원
- **시작 진단**: "연필 길이 어림하기" 1문제 + cm 자로 측정 1문제
- **종료 평가**: 4문제 (어림 1, 측정 1, 환산 1, 응용 1)
- **세션 패턴**:
  - 워밍업: 6단 자동화
  - 본문제: 실제 자로 재기, 어림 후 실측 비교, 단위 환산, 길이 비교 문장제
  - 도전: "교실 가로가 약 8m라면 책상 몇 개를 가로로 놓을 수 있을까?"

## 5. 구구단 6·7·8·9단 + 곱셈 응용

- **기간**: 3주 (6~7세션)
- **핵심 개념**: 6·7·8·9단 자동화 + 곱셈표 빈칸 추론 + 곱셈 활용 + 곱셈구구 패턴(짝수단·홀수단·9단 마법)
- **학교 진도 sync**: 2학기 곱셈구구 선행 (3Q 핵심 단원)
- **시작 진단**: 6단 5문제 (자동화 정도 측정)
- **종료 평가**: 5문제 (구구단 자동화 + 곱셈표 추론 + 활용 문장제)
- **세션 패턴**:
  - 워밍업: 단계별 약한 단 빠른 응답 (단계마다 신규 단 도입)
  - 본문제: 곱셈표 빈칸 채우기, "□ × 7 = 56" 류, 묶음 응용, 9단 손가락 트릭 도입
  - 도전: 곱셈구구 패턴 발견 ("9단 답의 자릿수 합은 항상 9?")
