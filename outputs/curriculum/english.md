## 1. 1주차 — 단모음 a (도입)

- **기간**: 1주 (4세션, 1세션은 2026-05-01에 이미 진행됨)
- **트랙 A 핵심**: 알파벳 a가 내는 "애" 소리. cat·hat·bat·mat·sat·pan / bag·map·fan·cap·jam
- **트랙 B 테마**: 동물 (cat, dog 맛보기, pig 맛보기) / **자주 나오는 단어**: a, I, is, this
- **트랙 C 회화**: What's this? / It's a ___. / Yes·No.
- **시작 진단**: 이미 1세션에 진행됨 (cat·sit·hop 자력 읽기 시도)
- **종료 체크 기준**: a 소리 단어 6개 중 4개 자력 읽기 + "It's a cat" 따라 말하기 OK
- **세션 패턴**: 워밍업(지난 단어 3~4개) → 오늘의 소리 → 단어 카드 5~6개 → 듣기·따라 말하기 → 회화 놀이 → 읽기 → 쓰기 1~2단어

## 2. 2주차 — 단모음 a (정착)

- **기간**: 1주 (4세션)
- **트랙 A 핵심**: 단모음 a 응용 단어 확장. 새 단어를 자력 blending으로 읽는 경험 누적.
  - 새 단어 예: ham, ban, tan, lap, gap, tag
- **트랙 B 테마**: 동물 + 음식 (ham, jam, apple) / **자주 나오는 단어**: the, see (누적: a, I, is, this, the, see)
- **트랙 C 회화**: I see a ___. / Yes·No. (확장: "I see a hat and a bat.")
- **종료 체크 기준 (단모음 a 종료)**:
  - 카드 6개 중 처음 보는 a 단어 4개 자력 발음 OK
  - "I see a ___" 패턴 따라 말하기 OK
  - 통과 시 → 3주차 단모음 i 진입. 미통과 시 → 1세션 더 연장.
- **자신감 트래커**: "처음 보는 단어 자력 읽기" 누적 카운트 시작

---

## 3. 3주차 — 단모음 i (도입)

- **기간**: 1주 (4세션)
- **트랙 A 핵심**: 알파벳 i가 내는 "이" 소리 (한국어 "이"보다 짧고 살짝 입을 벌린 느낌)
  - sit, pig, big, hit, lid, kid / pin, fin, tin, dig, fig
- **트랙 B 테마**: 음식 + 색깔 (red, blue, yellow, green) / **자주 나오는 단어**: it, my (누적 추가)
- **트랙 C 회화**: What color? / It's red. / My ___ is red.
- **시작 진단**: 단모음 a 단어 3개 자력 읽기 (cat·hat·bag 같은 것 — 정착 확인)
- **종료 체크 기준**: i 소리 단어 6개 중 4개 자력 읽기 + 색깔 단어 4개 인지

---

## 4. 4주차 — 단모음 i (정착) + o 도입

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: 단모음 i 응용 + a/i 섞어 읽기 (cat vs sit, bag vs big)
  - 3~4세션: 단모음 o 도입 ("아" 소리, 한국어 "어"와 "아" 사이) — hop, dog, top, mom, hot
- **트랙 B 테마**: 색깔 + 가족 (mom, dad, brother, sister, baby) / **자주 나오는 단어**: have, has
- **트랙 C 회화**: I have a ___. / This is my ___. / She has a ___.
- **종료 체크 기준 (단모음 i 종료)**: i 단어 자력 4/6 OK + a/i 구분 OK
  - 통과 시 → 5주차 단모음 o 본격. 미통과 시 → i 1세션 더 연장하고 o는 다음 주.

---

## 5. 5주차 — 단모음 o (정착) + **1차 누적 진단**

- **기간**: 1주 (4세션)
- **트랙 A 핵심**: o 소리 단어 확장 (mop, fox, box, log, pop, rock)
- **트랙 B 테마**: 가족 + 학교 (book, pen, bag, desk, chair) / **자주 나오는 단어**: 누적 점검
- **트랙 C 회화**: Hello / Bye / Thank you / How are you?
- **시작 진단** (5주차 첫 세션): 누적 단어 무작위 10개 읽기 + 회화 표현 5개 인지
- **마일스톤**: **누적 단어 ~50, 1차 누적 진단**
  - 단모음 a/i 단어 자력 읽기 6/10 OK
  - 회화 표현 5개 중 3개 따라 말하기 OK
  - 자주 나오는 단어 5개 중 3개 인지
- **세션 패턴**: 진단(5주차 1세션) → 단모음 o 누적 단어 노출 → 가족·학교 테마 카드 → 회화 패턴 반복

## 6. 6주차 — 단모음 o 종료 + u 도입

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: 단모음 o 마무리 + a/i/o 섞어 읽기 게임
  - 3~4세션: 단모음 u 도입 ("어"에 가까운 소리) — cup, sun, bus, run, fun, mud
- **트랙 B 테마**: 학교 + 몸 (head, hand, eye, ear, nose, mouth) / **자주 나오는 단어**: are, in, on
- **트랙 C 회화**: Where is ___? / Here it is. / It's on the ___.
- **종료 체크 기준 (단모음 o 종료)**: o 단어 자력 4/6 OK + a/i/o 섞어 읽기 6/9 OK

---

## 7. 7주차 — 단모음 u (정착)

- **기간**: 1주 (4세션)
- **트랙 A 핵심**: u 소리 단어 확장 (hug, bug, cut, nut, gum, jug)
- **트랙 B 테마**: 몸 + 옷 (hat, shirt, pants, shoes, socks) / **자주 나오는 단어**: 누적 점검
- **트랙 C 회화**: Put on your ___. / Take off your ___. / I wear ___.
- **세션 패턴**: 워밍업(누적 단어 4개) → u 단어 카드 → 옷 테마 카드 → "Put on/Take off" 동작 놀이 → 짧은 읽기

---

## 8. 8주차 — 단모음 u 종료 + e 도입

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: 단모음 u 마무리 + 4개 단모음(a/i/o/u) 섞어 읽기
  - 3~4세션: 단모음 e 도입 ("에" 소리) — bed, hen, pen, red, leg, ten
- **트랙 B 테마**: 옷 + 날씨 (sunny, rainy, cloudy, hot, cold) / **자주 나오는 단어**: can
- **트랙 C 회화**: How is the weather? / It's sunny today.
- **종료 체크 기준 (단모음 u 종료)**: u 단어 자력 4/6 OK + 4단모음 섞어 읽기 6/8 OK

---

## 9. 9주차 — 단모음 e (정착) + **2차 누적 진단**

- **기간**: 1주 (4세션)
- **트랙 A 핵심**: e 소리 단어 확장 (vet, jet, met, wet, ten, web) + 5단모음 종합 노출
- **트랙 B 테마**: 날씨 + 숫자 (one~ten) / **자주 나오는 단어**: 누적 점검
- **트랙 C 회화**: How many ___? / I have two ___. / I see five ___.
- **시작 진단** (9주차 1세션): 누적 단어 15개 무작위 읽기 + 회화 표현 8개 인지
- **마일스톤**: **누적 단어 ~75, 2차 누적 진단**
  - 단모음 a/i/o/u 단어 자력 읽기 9/15 OK
  - 회화 표현 8개 중 5개 따라 말하기 OK
  - "What's this? / It's a ___" 류 자유 응답 1회 이상

---

## 10. 10주차 — 단모음 5개 종합 + 자음 짝꿍 sh

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: 단모음 5개(a/i/o/u/e) 섞어 읽기 게임. 여기까지가 단모음 정착 종료점.
  - 3~4세션: 자음 짝꿍 sh 도입 ("쉬" 소리) — ship, shop, fish, dish, shell
- **트랙 B 테마**: 숫자 + 자연 (tree, flower, sun, moon, star, sky) / **자주 나오는 단어**: like
- **트랙 C 회화**: Look! / Wow! / I like ___. / I don't like ___.
- **종료 체크 기준 (단모음 5개 종료)**:
  - 5단모음 섞어 단어 12개 중 8개 자력 읽기 OK
  - 통과 시 → 11주차 ch/th 진입. 미통과 시 → 부족한 단모음 1세션 보강.

---

## 11. 11주차 — 자음 짝꿍 ch / th

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: ch ("취" 소리) — chip, chin, chop, lunch, much
  - 3~4세션: th ("ㄸ" 또는 "ㄷ" 비슷한 혀끝 소리) — this, that, the, three, thin
- **트랙 B 테마**: 자연 + 일상 동작 (run, jump, walk, sit, stand, eat, drink) / **자주 나오는 단어**: here, there
- **트랙 C 회화**: I can run. / Let's go. / Let's eat.
- **세션 패턴**: 워밍업(단모음 누적) → 짝꿍 소리 시범(시간 길게) → 단어 카드 → 동작 놀이 → 짧은 문장 읽기

---

## 12. 12주차 — 모음 짝꿍 ai / ay (맛보기) + 종합

- **기간**: 1주 (4세션)
- **트랙 A 핵심**:
  - 1~2세션: ai ("에이" 소리) — rain, snail, mail, train, paint
  - 3~4세션: ay ("에이" 소리, 단어 끝) — day, play, say, may, way
  - ai/ay는 같은 소리지만 위치만 다름을 자연스럽게 노출 (규칙화 X)
- **트랙 B 테마**: 일상 동작 + 감정 (happy, sad, angry, tired, hungry) / **자주 나오는 단어**: how, what
- **트랙 C 회화**: How are you? / I'm happy. / I'm tired. / I'm hungry.
- **마일스톤**: **누적 단어 100+ 도달**
- **종료 평가** (12주차 마지막 세션, 종합 진단):
  - 단모음 5개 단어 무작위 10개 자력 읽기 (목표 7/10)
  - sh/ch/th 단어 6개 중 4개 자력 읽기
  - ai/ay 단어 4개 중 2개 자력 읽기
  - 회화 표현 15개 중 10개 따라 말하기
  - 자주 나오는 단어 누적 15개 중 10개 인지
  - 자력 첫 읽기 누적 카운트 점검 (12주간 누적)

## 운영 메모

### english-tutor.md 출력 형식과 정합성
- 매 세션 출력은 진단·워밍업·오늘의 소리·단어 카드·듣기/따라 말하기·회화 놀이·읽기·쓰기·부모 발음 노트·마무리 구조 유지
- 단어 카드는 **이모지 + 단어 + 한국어 발음 + 뜻** 4열 필수
- 부모 자료에 전문용어(파닉스, vowel, blending, sight word, CVC 등) 절대 금지

### 진도 유연성 (D17 원칙 강하게 반영)
- **새 패턴은 한 세션 1개만**. 못 따라오면 같은 단계 1~2세션 더 연장.
- 단모음 한 개당 6~8세션 들 수 있다고 가정 (계획상 8세션이지만 더 늘어나도 OK).
- 12주 안에 단모음 5개까지만 가도 충분. ai/ay·sh/ch/th는 보너스 트랙.
- 매 단계 종료 체크 미통과 → 다음 단계 진입 보류, 같은 단계 보강 세션 추가.

### 자신감 트래커 운영
- 세션 결과(`/done`) 보고 시 **"처음 보는 단어를 아이가 자기 힘으로 읽은 횟수"** 누적
- 50단어 자력 읽기 달성 / 100단어 자력 읽기 달성 마일스톤 기념
- 막힌 단어는 카운트하지 않고, 부모가 즉시 알려준 뒤 다음 세션 워밍업에 다시 노출

### 미니 누적 진단 (5주차·9주차)
- **5주차 진단**: 누적 단어 무작위 10개 자력 읽기 + 회화 표현 5개 인지 + 자주 나오는 단어 5개 인지
- **9주차 진단**: 누적 단어 15개 자력 읽기 + 회화 표현 8개 인지 + 자유 응답 1회 이상
- 진단 후 약한 영역(소리 / 단어 / 회화 / 자주 나오는 단어)에 다음 1주 비중 조정

### 단어 누적 트래커
- 매 주 종료 시 누적 단어 수를 `outputs/progress.md`에 기록
- 마일스톤: 25 / 50 / 75 / 100 단어 도달 시점에 짧은 칭찬 포인트
- 단어 = "이모지 보고 영어로 말할 수 있는 단어" 기준 (자력 발음과 별도로 카운트)

### 입력(Input) 노출 보강
- 부모 발음 시범 시간을 짧게라도 매 세션 확보 (3~5번 반복 청취)
- 가능하면 세션 외 시간에 영어 노래·짧은 영상 1~2분 노출 권장 (강제 X, 흥미 트리거)
