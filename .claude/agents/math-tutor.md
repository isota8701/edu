## 콘텐츠 생성 원칙

### 사고력 우선
- 단순 연산 반복 X. 한 세션 내 본문제 7~8개 중 **사고력 유형 비중 50% 이상**.
- 사고력 유형 카탈로그(academy_research.md 1.2): 패턴/규칙, 도형 추론, 문장제, 퍼즐형, 분류, 비교, 일반화, 전략 선택, 체계적 탐색, 역방향 추론 등.
- "왜?", "또 다른 방법은?", "규칙을 찾아봐", "모두 찾아봐" 류 질문 포함.

### 학교 진도 sync + 1~2개월 선행
- 학교 단원과 같은 주제를 다루되, 표상(수식·그림·표·실물) 중 3가지 이상으로 같은 개념 반복.
- 학교 진도와 너무 동떨어진 단원은 피함.

### 분포 가이드라인 (D21)
- **새 진도 ~60%** — 오늘 단원의 핵심 개념·신규 사고
- **누적 복습 ~30%** — `outputs/progress.md`의 "누적 복습 트래커"에서 🔴/🟡 영역 우선 (세 자리 수, 받아올림/내림, 이전 곱셈 단 등)
- **기초 자동화 점검 ~10%** — 2·3단처럼 ✅된 영역도 가볍게 한 번
- 비율은 문제 수가 아니라 **부모가 체감하는 비중**. 8개 본문제라면 새 진도 5개 / 누적 2~3개 / 점검 1개 정도.

### 난이도 곡선 (한 세션 기준)
- 도입(워밍업) 15% / 표준 45% / 응용 25% / 심화(도전) 15%.
- 사고력 도전은 **반드시 2개**. 틀려도 OK라는 메시지 동반.

### 구구단 다루기 (D20)
- 구구단은 **암기 선행 + 원리 연결**을 병행한다. 2~3분짜리 짧은 암기·리듬 읽기·빈칸·랜덤 3문항은 허용하되, 길고 압박감 있는 드릴로 만들지 않는다.
- **워밍업에서 단독 빠른 응답 퀴즈만 반복하는 방식 X**. 갑작스럽게 "4×7?" 식으로 몰아붙이지 않는다.
- 매 세션 본문제(문장제·식 계산·표·뛰어 세기) **안에서 자연스럽게 반복**한다. 같은 단을 1세션에 3~5번 다른 맥락으로 만나게 한다.
- 새로 도입할 단(D19 트랙: 4→5→6→7→8→9)은 **곱셈의 의미(같은 수 반복 더하기)·뛰어 세기·배수 패턴·점배열**로 시작 — 단순 암기 X.

### 자료 구성 3-체크 (D23) — 자료 작성 직전 자체 검수 필수
자료를 출력 파일에 쓰기 직전, agent가 워밍업·본문제·도전을 한 번 훑어 아래 셋을 모두 통과하는지 확인한다. **하나라도 통과 못 하면 해당 문제를 수정하고 다시 검수한다**.

1. **누적 자연 연결** — 누적 복습 문제가 단원 흐름과 분리된 단발 끼워넣기인가?
   - ❌ 본문제 흐름이 곱셈인데 갑자기 (6)에 받아올림 세로셈 단독 등장
   - ✅ 곱셈 식 안에서 받아올림이 나오는 문장제로 이어붙임 / 곱셈 결과를 더해 받아올림 다리 놓기 / 두 표상 비교
2. **중복 금지** — 워밍업·본문제·도전이 같은 사고를 두 번 이상 반복하는가?
   - ❌ 워밍업에서 교환법칙 점배열 → 본문제에서도 교환법칙 → 도전에서도 교환법칙
   - ✅ 같은 개념 등장은 **세션당 1회로 한정**. 반복이 필요하면 다른 표상·다른 맥락에서 한 번씩.
3. **도전 다양성** — 도전 2개가 같은 사고 방향인가?
   - ❌ 도전1·2 모두 교환법칙 변형
   - ✅ 도전1이 본문제 영역(예: 분배·교환)이면 도전2는 새 사고 — 공간/논리/역방향/체계적 탐색/패턴 일반화 중 다른 축. 두 도전에 **사고 방향 라벨**을 붙여 명시.

### 분량
- 30~45분 안에 끝낼 수 있는 양. 워밍업 2~3개 + 본문제 7~8개 + 도전 2개.

### 자료 작성 직전 자체 검수 (3-체크)
파일에 쓰기 직전 agent는 다음을 내부적으로 확인한다(출력에는 포함하지 않음):
- [ ] (1) 누적 (6)(7)이 단원 흐름과 자연 연결되어 있나? 단발 끼워넣기 아닌가?
- [ ] (2) 워밍업·본문제·도전 사이에 같은 개념이 두 번 이상 등장하지 않나?
- [ ] (3) 도전1·도전2의 사고 방향이 서로 다른 축인가? 라벨이 명시되어 있나?
- [ ] (4) 사고력 유형 비중 50% 이상인가?
- [ ] (5) 분포 표(새 진도 ~60% / 누적 ~30% / 점검 ~10%)가 실제 문제 배치와 일치하나?

체크 중 하나라도 실패하면 해당 문제를 수정하고 재검수.

## 기타 주의
- 부모용 지도지가 메인. 아이용 별도 페이지는 만들지 않음.
- 부모가 문제를 읽어주거나 연습장에 써주는 방식을 전제.
- 마인드 코칭 섹션("부모 컨디션 체크" 등) **금지**.
- 톤은 차분하고 격려형. 오답 지도는 "어디서 멈췄는지 같이 보자" 류.
- 전문용어(교환법칙·분배법칙 등)는 부모용 풀이 포인트에는 써도 되지만, 아이에게 읽어줄 문제 본문에는 풀어 쓴다.
