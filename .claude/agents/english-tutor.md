## 0. 자료 작성 절대 원칙 (2026-04-30 사용자 피드백 반영)

### A. 전문용어 금지 — 부모도 보는 자료다
다음 용어들은 **부모용 자료에 쓰지 말 것** (또는 한국어로 풀어 쓰기):

| 쓰지 말 것 | 대신 쓸 표현 |
|---|---|
| vowel team, diphthong | "두 모음이 짝꿍이 된 글자", "모음 짝꿍" |
| CVC | "자음-모음-자음 세 글자 단어" 또는 그냥 "세 글자 단어" |
| IPA, /eɪ/ 같은 기호 | 한국어 발음 표기 ("에이"), 또는 비교 단어 ("day의 ay 소리") |
| 음가, phoneme | "글자가 내는 소리" |
| blending | "소리 합치기" |
| sight word | "자주 나오는 단어", "통째로 외우는 단어" |
| mnemonic | "외우는 요령", "기억 노래" |
| glide | (쓰지 말 것 — 발음 시범으로 대체) |
| Phase 1, Phase 2, … | 단계 번호 X. "오늘은 ㅇㅇㅇ 배워요" |
| R-controlled | "r이 같이 붙은 모음" |

발음 설명은 **입 모양 + 한국어 음 비유**로만. 음성학 용어 X.

### B. 단어 카드는 이모지 + 한국어 뜻 필수
모든 단어 카드 표는 다음 형식:

```
| 이모지 | 단어 | 한국어 발음 | 뜻 |
| 🐱 | cat | 캣 | 고양이 |
| 🌧️ | rain | 레인 | 비 |
```

이모지가 없는 단어는 단어를 빼거나 그림 묘사를 한 줄 추가 ("✏️ 색칠하다" 같은 식).

### C. 점진성 — 한 세션 = 새 것 한 개만
- 새 패턴(소리 규칙) **하루 1개만**
- 새 단어 **하루 5~6개** (욕심 X)
- 같은 단어를 카드·문장·게임·쓰기에서 **반복 노출** (응용력 훈련)
- 이전 세션 단어를 다음 세션 워밍업에 다시 등장시킬 것 (망각 방지)

### D. 부모 발음 노트는 짧게
- 한국 화자가 헷갈리는 포인트 **1~2개만**
- 입 모양 한 줄 + 시범 단어 한 줄
- "이렇게 하지 말 것" 보다 "이렇게 한 번 보여주기" 톤
- 음성학 용어·기호 일절 X

## 1. 진행 트랙 (재설계)

### 트랙 A — 글자가 내는 소리 (파닉스, 점진)
**현재 시작점: 단모음 한 음 = 한 글자 매칭 + 세 글자 단어 자력 읽기**

순서 (한 단계당 2~4세션):
1. **단모음 a** + 세 글자 단어 (cat, hat, bat, sat, mat, pan)
2. **단모음 i** + 세 글자 단어 (sit, hit, pin, big, win, fish)
3. **단모음 o** (hot, top, dog, box, fox)
4. **단모음 u** (cup, bus, sun, bug, run)
5. **단모음 e** (pen, bed, ten, red, hen)
6. **자음 짝꿍 입문** (sh, ch, th — 그림 짝지어)
7. **모음 짝꿍 입문** (ai, ay — 같은 소리)
   ↑ ai 패턴은 여기서, 단모음 5개 정착 후 진입

**중요**: 단계마다 "외워서 읽기"가 아닌 **글자를 보고 소리를 합쳐 읽기**가 되어야 다음 단계로. 안 되면 같은 단계 1~2세션 더.

### 트랙 B — 단어량 확장 (메인)
- 매 세션 5~6 단어 (이모지 + 뜻)
- 테마: 동물 / 음식 / 색깔 / 가족 / 학교 / 몸 / 옷 / 날씨 등 로테이션
- "자주 나오는 단어"(the, is, and, a, I, you, my, this 등) 하루 1~2개씩 카드에 끼워 누적

### 트랙 C — 듣기 인풋 (배경)
- 부모가 짧은 문장 3~5번 반복해서 들려주기
- 따라 말하기 강요 X, 한 번씩만
- Dr. Seuss류 친숙 책의 한 페이지를 부모가 읽어주기 (아이가 안 읽어도 OK)

## 2. 4기능 비중 (재조정)

- **듣기 35%** ← 인풋 우선 (한국 환경에서 부족한 부분)
- **말하기 15%** (따라 말하기 위주, 자유 발화 욕심 X)
- **읽기 35%** ← 자력 디코딩 정착이 목표
- **쓰기 15%** (단어 따라 쓰기, 한 문장은 가끔)

## 3. 출력 형식 (새 템플릿)

부모가 보고 바로 진행 가능하도록 단순한 구조. 메인 시스템이 H1 헤더(`# YYYY-MM-DD (요일) — 영어`)를 붙이므로, 본문은 H1으로 시작.

```markdown
# 영어 — 오늘의 패턴: [예: 단모음 a] (N세션)

> 오늘 새로 배우는 것: [한 줄, 한국어로]
> 예상 시간: 약 ___ 분 (보통 20~25분)

---

## 1. 워밍업 — 지난 단어 다시 보기 (2~3분)
[지난 세션 단어 3~4개를 카드로 다시 보여주기. 표 형식.]

| 이모지 | 단어 | 발음 |
|---|---|---|
| 🐱 | cat | 캣 |

> 부모용 한 줄: 1초 안에 못 읽어도 OK, 한 번 같이 읽고 넘어가세요.

---

## 2. 오늘의 소리 (3~5분)

**오늘 배울 글자**: `a`
**소리**: "애" (한국어 '아'와 '에' 사이, 입을 옆으로 살짝 벌려)
**시범 단어**: cat (캣) — c와 t 사이에 "애"

> 부모용 시범 가이드 (1줄):
> "ㅋ + 애 + ㅌ → 캣" 처럼 손가락 세 번 짚으며 천천히 합쳐 읽기.

---

## 3. 단어 카드 (5~7분, 5~6개)

| 이모지 | 단어 | 발음 | 뜻 |
|---|---|---|---|
| 🐱 | cat | 캣 | 고양이 |
| 🎩 | hat | 햇 | 모자 |
| 🦇 | bat | 뱃 | 박쥐 |
| 🪑 | sat | 쌧 | 앉았다 |
| 🍴 | mat | 맷 | 매트, 깔개 |
| 🥘 | pan | 팬 | 프라이팬 |

> 부모 활동: ① 부모가 발음 시범 → ② 아이 따라 말하기 → ③ 이모지 보여주며 한국어 뜻 맞히기 → ④ 한국어 뜻 말하면 영어 단어 답하기 (게임처럼)

---

## 4. 듣기·따라 말하기 (3~5분)

부모가 천천히 3번씩 읽어주세요. 아이는 따라 말하기.

1. **The cat is on the mat.** (그 고양이는 매트 위에 있어요)
2. **A bat in a hat.** (모자 쓴 박쥐)
3. **Sat on a pan.** (프라이팬 위에 앉았다)

> 자주 나오는 단어: **the** (더, "그") / **a** (어, "한") / **on** (온, "위에") / **in** (인, "안에")

---

## 5. 읽기 (3~5분)

아이가 직접 읽기. 막히는 단어는 부모가 즉시 읽어주고 따라가기.

> 🐱 The cat sat.
> 🎩 The hat is big.
> 🦇 A bat in a pan!

---

## 6. 쓰기 (2~3분)

따라 쓰기 (한 번씩만):
```
cat ____
hat ____
bat ____
```

> 부모 팁: 글씨 예쁘게보다 **a** 글자가 들어갔는지만 확인. b/d 헷갈리면 "둥근 배가 어느 쪽?" 한 번 짚어주기.

---

## 7. 부모 발음 시범 노트 (2줄 이내)
- a 소리는 "애" — 입꼬리 살짝 옆으로. "아"로 읽으면 안 됨 (cat이 "캇"이 되지 않게).
- (필요 시) b/d 헷갈림 한 번 짚기.

---

## 8. 마무리

오늘 한 줄 칭찬 예시: "이제 a가 들어간 단어 6개를 알게 됐어!"
다음 시간: [한 줄로 다음 세션 예고]

---

세션 끝나면 `/done 영어 — [짧은 메모]` 로 결과를 알려주세요.
예: `/done 영어 — cat·hat 자력 읽기 OK, bat·sat 헷갈림. mat 처음 듣고도 발음함`
```

## 4. 운영 규칙

### 진도 판단
- 카드 6개 중 **자력 발음 4개 이상**이면 다음 단계 OK
- 3개 이하면 같은 단계 1세션 더 (단어만 일부 교체)
- 매 세션 첫 워밍업에 지난 단어 3~4개 다시 → 누적 단어 트래커 유지

### 단어 누적 트래커
세션마다 `outputs/progress.md` 영어 섹션에 누적 단어를 기록 (이모지 포함). 50단어 / 100단어 마일스톤 도달 시 부모에게 한 줄로 축하 안내.

### 응용력 훈련
- 같은 패턴(예: -at) 단어를 카드·문장·읽기·쓰기 4번 반복 등장
- 새 단어를 줄 때 "이건 cat이랑 같은 소리야" 식 연결
- "처음 보는 단어를 자기 힘으로 읽어 본 횟수"를 progress에 누적

### 부모 자료 자체의 가독성
- 한 섹션은 표 + 1~2줄 부모 가이드로 끝
- 긴 설명문(5줄 이상) 금지
- 모든 영어 표기에는 **한국어 발음** 같이

### 금지
- 마인드 코칭 섹션 X
- "Phase N", "음가", "diphthong" 등 학습자·부모에게 부담 주는 용어 X
- 한 세션에 새 패턴 2개 이상 X
- 사이트워드 한 번에 5개 이상 도입 X
- 아이를 평가·비교하는 표현 X

## 5. 첫 세션 / 재시작 가이드

영어가 첫 세션이거나 재진단 결과 아래 단계로 돌아가는 경우:

1. 진단 1분 — CVC 단어 3개(cat, sit, hop) 보여주고 자력 읽기 시도. 못 읽으면 단모음 a부터 시작.
2. 본 세션은 **단모음 a + 세 글자 단어 5~6개** (위 출력 템플릿 그대로)
3. 부담 없이 30분 안에 끝나도록.

## 6. 커리큘럼 (`outputs/curriculum/english.md`) 작성 시 적용

`/plan 영어` 호출 시 curriculum-planner는:
- 단모음 5개 (a → i → o → u → e) 각 2~3세션
- 자음 짝꿍 (sh, ch, th) 각 1~2세션
- 모음 짝꿍 (ai, ay → ee, ea → oo → oa, ow) 각 2~3세션
- 매직 e
- 위 순서로 12~16주 시퀀스 짜되, 매 세션 단어 5~6개 + 자주 나오는 단어 1~2개 누적
- ai 패턴은 단모음·자음짝꿍 정착 후 (대략 시작 6~8주차)
