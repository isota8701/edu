## 0. 자료 작성 절대 원칙 (2026-04-30 사용자 피드백 + 2026-05-07 D22 반영)

### A. 전문용어 금지 — 부모도 보는 자료다
다음 용어들은 **부모용 자료에 쓰지 말 것** (또는 한국어로 풀어 쓰기):

| 쓰지 말 것 | 대신 쓸 표현 |
|---|---|
| vowel team, diphthong | "두 모음이 짝꿍이 된 글자", "모음 짝꿍" |
| CVC | "자음-모음-자음 세 글자 단어" 또는 그냥 "세 글자 단어" |
| IPA, /eɪ/ 같은 기호 | 한국어 발음 표기 ("에이"), 또는 비교 단어 ("day의 ay 소리") |
| 음가, phoneme | "글자가 내는 소리" |
| blending | "소리 합치기" |
| sight word | "자주 나오는 단어", "통째로 외우는 단어" |
| mnemonic | "외우는 요령", "기억 노래" |
| glide | (쓰지 말 것 — 발음 시범으로 대체) |
| Phase 1, Phase 2, … | 단계 번호 X. "오늘은 ㅇㅇㅇ 배워요" |
| R-controlled | "r이 같이 붙은 모음" |

발음 설명은 **입 모양 + 한국어 음 비유**로만. 음성학 용어 X.

### B. 단어 카드는 이모지 + 한국어 뜻 필수
모든 단어 카드 표는 다음 형식:

```
| 이모지 | 단어 | 한국어 발음 | 뜻 |
| 🐱 | cat | 캣 | 고양이 |
| 🌧️ | rain | 레인 | 비 |
```

이모지가 없는 단어는 단어를 빼거나 그림 묘사를 한 줄 추가 ("✏️ 색칠하다" 같은 식).

### C. 점진성 + 누적 회전 (D22 갱신)
- 새 패턴(소리 규칙) **하루 1개만**
- **신규 단어 하루 2~3개** (D22 — 종전 5~6개에서 축소). 단, **단어량 자체는 계속 늘림** — 같은 단어 무한 반복은 솔루션 X.
- **누적 단어 회전 필수**: 정착 안 된 단어는 매 세션 워밍업·문장·회화·게임 어딘가에 반드시 등장. 정착된 단어는 가끔 점검.
- 같은 단어를 카드·문장·게임·쓰기에서 **반복 노출** (응용력 훈련)
- 사이트워드는 **트랙 E**(아래)로 별도 누적, 한 번에 5개 이상 신규 도입 X

### D. 부모 발음 노트는 짧게
- 한국 화자가 헷갈리는 포인트 **1~2개만**
- 입 모양 한 줄 + 시범 단어 한 줄
- "이렇게 하지 말 것" 보다 "이렇게 한 번 보여주기" 톤
- 음성학 용어·기호 일절 X

## 1. 진행 트랙 (재설계)

### 트랙 A — 글자가 내는 소리 (파닉스, 점진)
**현재 시작점: 단모음 한 음 = 한 글자 매칭 + 세 글자 단어 자력 읽기**

순서 (한 단계당 2~4세션):
1. **단모음 a** + 세 글자 단어 (cat, hat, bat, sat, mat, pan)
2. **단모음 i** + 세 글자 단어 (sit, hit, pin, big, win, fish)
3. **단모음 o** (hot, top, dog, box, fox)
4. **단모음 u** (cup, bus, sun, bug, run)
5. **단모음 e** (pen, bed, ten, red, hen)
6. **자음 짝꿍 입문** (sh, ch, th — 그림 짝지어)
7. **모음 짝꿍 입문** (ai, ay — 같은 소리)
   ↑ ai 패턴은 여기서, 단모음 5개 정착 후 진입

**중요**: 단계마다 "외워서 읽기"가 아닌 **글자를 보고 소리를 합쳐 읽기**가 되어야 다음 단계로. 안 되면 같은 단계 1~2세션 더.

### 트랙 B — 단어량 확장 (메인, D22 조정)
- 매 세션 **신규 2~3 단어** (이모지 + 뜻) — 신규 페이스만 줄임
- **누적 단어 풀**은 계속 커짐. 매 세션 워밍업/문장/회화에서 누적 단어 회전
- 테마: 동물 / 음식 / 색깔 / 가족 / 학교 / 몸 / 옷 / 날씨 등 로테이션
- 정착 미달 단어(자력 발음 못 하는 것)는 다음 세션에 반드시 다시 등장

### 트랙 C — 듣기 인풋 (배경)
- 부모가 짧은 문장 3~5번 반복해서 들려주기
- 따라 말하기 강요 X, 한 번씩만
- Dr. Seuss류 친숙 책의 한 페이지를 부모가 읽어주기 (아이가 안 읽어도 OK)

### 트랙 D — 자음 혼동 미니 게임 (D22 신설, 매 세션 2~3분)
b/d, f/p, m/n, p/q 등 글자 모양·소리 헷갈리는 자음을 짧게 다룬다.
- 시작: **b/d**, **f/p** (5/4 세션에서 발견된 약점)
- 형식: 글자 모양 짚기(예: "b는 동그란 배가 오른쪽") + 부모 발음 시범 + 짧은 분류 게임 (4~6장 카드 어느 글자로 시작?)
- 한 세션 1쌍만 (b/d 한 세션, 다음 세션 f/p 식으로 로테이션)
- 정착되면 m/n, p/q, c/k 등으로 확장

### 트랙 E — 사이트워드 카드 (D22 신설, 매 세션 3~5장)
"자주 나오는데 소리 규칙으로 안 읽히는 단어"는 통째로 외우는 카드로.
- 시작 묶음: **is, this, the, a, I, see, on, with** (5/4에서 막혔던 is·this 우선)
- 신규는 한 세션 1~2개만 추가, 나머지 3~4장은 **누적 카드 회전**
- 형식: 단어 + 한국어 발음 + 짧은 예시 ("is" → "이즈" → "It is a cat.")
- 한 번에 5개 이상 신규 도입 X — 천천히 쌓기

### 트랙 F — 맥락형 회화 (D22 신설/확대, 매 세션 1상황)
"어떤 때 쓰는 말인지" 한국어 안내와 함께 1상황 표현 1~2개.
- 상황 로테이션: 인사 / 요청 / 놀이 / 식사 / 학교 / 칭찬 / 사과 / 감정 표현
- 형식: "이 표현은 ㅇㅇ할 때 써요" 한국어 한 줄 → 영어 표현 1~2개 → 부모-아이 짧은 주고받기 2턴
- 누적 표현은 다음 세션 워밍업이나 듣기 문장에 다시 등장

## 2. 자음 혼동 미니 게임 (2~3분) — 트랙 D
**오늘의 헷갈림 쌍**: b vs d
- **b**: 동그란 배가 **오른쪽** — "ㅂ" 입술 다물었다 떼는 소리
- **d**: 동그란 배가 **왼쪽** — "ㄷ" 혀끝 윗니 뒤
- 시범 단어: **b**at (뱃) / **d**ad (대드)

> 게임: 카드 4장(bat, dad, bag, dog) — 어느 글자로 시작? 손가락으로 짚기.

## 3. 오늘의 소리 (3~4분) — 트랙 A

**오늘 배울 글자**: `a`
**소리**: "애" (한국어 '아'와 '에' 사이, 입을 옆으로 살짝 벌려)
**시범 단어**: cat (캣) — c와 t 사이에 "애"

> 부모용 시범 가이드 (1줄):
> "ㅋ + 애 + ㅌ → 캣" 손가락 세 번 짚으며 천천히 합쳐 읽기.

## 4. 새 단어 카드 (4~5분, 신규 2~3개) — 트랙 B
신규는 적게, 대신 위 워밍업 단어와 합쳐 게임처럼.

| 이모지 | 단어 | 발음 | 뜻 |
|---|---|---|---|
| 🥘 | pan | 팬 | 프라이팬 |
| 🎒 | bag | 백 | 가방 |
| 🗺️ | map | 맵 | 지도 |

> 부모 활동: ① 발음 시범 → ② 아이 따라 → ③ 이모지 보여주며 뜻 맞히기 → ④ 뜻 말하면 영어 답하기

---

## 5. 사이트워드 카드 (3~4분, 3~5장) — 트랙 E
통째로 외우는 단어. 신규 1~2장 + 누적 2~3장.

| 단어 | 발음 | 짧은 예 |
|---|---|---|
| is | 이즈 | It **is** a cat. |
| this | 디스 | **This** is a bag. |
| the | 더 | **the** map |
| a | 어 | **a** pan |

> 부모용 한 줄: 손가락으로 한 글자씩 짚지 말고 **단어 통째**로 보여주세요. 소리 합치기 X.

---

## 6. 맥락 회화 (3~5분, 1상황) — 트랙 F
**오늘 상황**: [예: 누가 뭘 가지고 있는지 묻기]
**어떤 때 쓰나요?**: 친구가 가진 물건을 가리키며 "이게 뭐야?" 물을 때

| 표현 | 발음 | 뜻 |
|---|---|---|
| What is this? | 왓 이즈 디스? | 이게 뭐야? |
| It is a bag. | 잇 이즈 어 백 | 가방이야. |

> 주고받기 2턴: 부모가 물건 들고 "What is this?" → 아이 "It is a ___." → 역할 바꾸기.
> 누적: 지난 세션에 배운 표현 1개 다시 한 번 ([예: "Hello, how are you?"])

## 7. 듣기·따라 말하기 (3~4분)

## 4. 운영 규칙 (D22 갱신)

### 진도 판단
- **신규 2~3개 + 워밍업 4~5개 = 한 세션 단어 노출 6~8개**
- 신규 2~3개 중 **자력 발음 2개 이상**이면 패턴 진도는 OK 신호
- 워밍업 누적 단어 정착도가 60% 미만이면 같은 패턴 1세션 더 (신규 페이스 0~1개로 줄여 누적 회전 강화)
- **정착 미달 단어**는 다음 세션 워밍업/문장/회화에 반드시 재등장 (망각 방지)

### 트랙 D(자음) 진도
- 한 쌍(예: b/d)이 4장 중 3장 이상 정답이면 다음 쌍으로 로테이션
- 정착돼도 2~3세션마다 한 번씩 점검 카드로 재등장

### 트랙 E(사이트워드) 진도
- 누적 카드 중 **즉시 읽기 80%** 도달 시 신규 1~2장 추가
- 한 세션 신규 1~2장만, 절대 5장 이상 X

### 트랙 F(회화) 진도
- 한 상황 표현이 부모 질문에 자력 응답 2/3 가능하면 다음 상황으로
- 누적 표현은 워밍업 또는 듣기 문장에 1주에 1회는 재등장

### 단어 누적 트래커
세션마다 `outputs/progress.md` 영어 섹션에 다음 4개 풀 기록:
- **단어 누적 풀** (이모지 포함, 정착/미달 표시)
- **사이트워드 누적 풀**
- **회화 표현 누적 풀** (상황 태그)
- **자음 혼동 점검 이력** (어떤 쌍 언제 통과)

50단어 / 100단어 마일스톤 도달 시 부모에게 한 줄 축하 안내.

### 응용력 훈련
- 같은 패턴(예: -at) 단어를 카드·문장·읽기·쓰기 4번 반복 등장
- 새 단어를 줄 때 "이건 cat이랑 같은 소리야" 식 연결
- "처음 보는 단어를 자기 힘으로 읽어 본 횟수"를 progress에 누적

### 부모 자료 자체의 가독성
- 한 섹션은 표 + 1~2줄 부모 가이드로 끝
- 긴 설명문(5줄 이상) 금지
- 모든 영어 표기에는 **한국어 발음** 같이

### 금지
- 마인드 코칭 섹션 X
- "Phase N", "음가", "diphthong" 등 학습자·부모에게 부담 주는 용어 X
- 한 세션에 새 패턴 2개 이상 X
- 한 세션 신규 단어 4개 이상 X (D22 — 2~3개 상한)
- 사이트워드 한 번에 5개 이상 도입 X
- 자음 혼동 미니 게임 한 세션 2쌍 이상 X (한 쌍씩만)
- 같은 단어만 무한 반복 X — 단어량은 계속 늘어가야 함
- 아이를 평가·비교하는 표현 X

## 5. 첫 세션 / 재시작 가이드

영어가 첫 세션이거나 재진단 결과 아래 단계로 돌아가는 경우:

1. 진단 1분 — CVC 단어 3개(cat, sit, hop) 보여주고 자력 읽기 시도. 못 읽으면 단모음 a부터 시작.

3. 부담 없이 30분 안에 끝나도록.

## 6. 커리큘럼 (`outputs/curriculum/english.md`) 작성 시 적용

`/plan 영어` 호출 시 curriculum-planner는:
- 단모음 5개 (a → i → o → u → e) 각 2~3세션
- 자음 짝꿍 (sh, ch, th) 각 1~2세션
- **모음 짝꿍·매직 e는 후순위로 이동** (D22) — 단모음·자음 혼동·사이트워드 안정 후 진입
- 모음 짝꿍 (ai, ay → ee, ea → oo → oa, ow) 각 2~3세션
- 매직 e
- 위 순서로 12~16주 시퀀스 짜되, **매 세션 신규 단어 2~3개 + 사이트워드 신규 1~2개 + 누적 회전** (D22)
- 매 세션 자음 혼동 미니 게임 1쌍, 맥락 회화 1상황 끼워넣기
- ai 패턴은 단모음·자음짝꿍·사이트워드 안정 후 (대략 시작 8~10주차로 후순위)
