### 외부 피드백 (수학 학원, 2026-05-12)
- 학원 선생님 평가: 유주는 **도형과 연산은 잘하는 편**.
- 보강 필요: **곱셈구구가 부족**. 구구단은 우선 외워 자동화하고, 이후 원리를 연결해 이해하는 흐름이 좋다는 조언.
- 두 자리 수 뺄셈은 집에서 힘들어하는 모습이 있었으나, 학원 관점에서는 연산이 전반적으로 **천천히 가는 편**에 가깝고, 연습량이 늘면 좋아질 것으로 봄.
- 해석: 정확도 자체를 낮게 보지 말고, **속도·자동화·즉시 인출**을 별도 목표로 둔다.

### 현재 수준
- 받아올림/받아내림: **정확도는 높음**. 학교 논술형 평가에서 86+37처럼 십의 자리·백의 자리까지 받아올림이 있는 계산, 17+26·21-4 블록 문제, 21+13-9 두 단계 문장제를 모두 맞힘. 다만 낯선 과정식 표상에서는 흔들릴 수 있음.
- 두 자리 수 덧셈·뺄셈은 세로셈·수 분해·역연산 관계·□식 만들기까지 노출됨. 2026-05-11 평가에서 덧셈식↔뺄셈식 관계(46+38=84 → 84-46=38, 84-38=46 / 80-43=37 → 43+37=80, 37+43=80), □+7=23, 10-□=4, □-5=7 유형을 모두 해결.
- **논술형 과정식 약점**: 처음 보는 표상인 "십 만들기 과정식"과 "계층 순서도형 중간값 기록"에서 오답. 예: 47+19 = 47+3+16 = 50+16 = 66, 42-17+37은 42-17=25 → 25+37=62처럼 중간값을 차례로 적는 구조를 보강해야 함.
- **식 전개 감각**: 2026-05-11 자료에서 69+18 = 69+1+17 = 70+17처럼 수를 갈라 가까운 십을 만드는 이유를 이해함. 단순 계산보다 "왜 이렇게 바꾸는가"를 명시하면 흡수 좋음.
- **문장→식 변환**: 계산은 맞지만, 비교 문장제에서 "35와 32 중 누가 몇 개 더 많은가"를 35-32로 쓸지 35+32로 쓸지 잠깐 고민. 문제 문장을 듣고 알맞은 연산을 고르는 훈련 필요.
- **구구단 자동화**: 학원 피드백상 부족. 문장제 맥락에서는 일부 해결하지만, 즉시 인출·랜덤 응답은 아직 약한 것으로 본다. 구구단은 우선 외워 입에 붙이고, 문제 안에서 원리를 연결해 이해하는 방식으로 병행.
- **구구단 4단**: 문장제 맥락 안에서는 OK (4×6=24 등). 단독 질문에서는 흔들림 있음.
- **구구단 5단**: 5×1~5 + 5×6~9 모두 워밍업·본문제에서 풀었으나, 자동화 완료로 보지 않고 반복 필요. 자기검산 도구로 끝자리 패턴(0/5) 인식 시작.
- **세 자리 수 자릿값 사고**: 2세션 안정 도달. 1단원 종료 (2026-05-01).
- **교환법칙**: 90도 회전 점배열 시각화로 확실히 이해 (2026-05-04). 추가 반복 X — 다른 사고 영역으로.
- 곱셈의 의미: 묶음 ↔ 식 변환, 빈칸 채우기 안정. 분배법칙 맛보기는 시점 빠름 — 별도 단원에서 다시.

### 보강 우선순위
1. **문장→식 변환** — "몇 개 더 많은가", "남은 것은", "모두 몇 개인가"를 듣고 `+`, `-`, `×` 중 어떤 식으로 바꿀지 고르는 훈련.
2. **식 전개 감각 유지** — 십 만들기(69+18 → 69+1+17 → 70+17), 중간값 기록, 곱셈 후 덧셈으로 이어지는 전개. 아이에게 "식을 보기 좋게 바꾸는 재미"로 접근하면 반응 좋음.
3. **체계적 탐색 능력** — "답이 N인 곱셈식 모두 찾기"에서 큰 인수 쌍은 유도 필요. 2×12 같은 구구단 밖 수는 힌트 표·반쪽 만들기 질문과 함께.
4. **구구단 암기·자동화** — 4·5단부터 짧게 매일 외워 입에 붙이고, 문제 안에서 원리와 연결. 갑작스러운 압박 퀴즈는 피하되, 리듬 읽기·빈칸 채우기·랜덤 3문항처럼 부담 낮은 즉시 인출은 포함.
5. 곱셈의 의미 — 묶음 모델 심화, 다양한 문장제 (교환법칙은 이미 안착, 반복 X)
6. **2·3단 점진적 본격화** — 가벼운 노출 끝나고 본문제 진입
7. 사고력 수학 — 패턴·추론·문장제 (✓ 반응 매우 좋음, 문제 수 더 늘릴 것). 도전 문제 2개는 **서로 다른 사고 방향**이어야 함.
8. 도형 직관 강화 (7단원에서)

### 세션 설계 원칙 (부모 피드백 반영)
- **구구단**: 매 세션 문제 안에서 자연스럽게 — 갑작스러운 단독 질문 방식 X
- **구구단 암기 순서**: 학원 피드백을 반영해 "일단 외우기 → 문제 속 원리 이해" 흐름을 허용. 단순 드릴만 길게 하지 말고 2~3분 짧은 자동화 루틴으로 운영.

## 영어

### 현재 수준 (2026-05-04 2세션 후 재확인)
- **읽기**: CVC 단어도 자력으로 소리 합치기 매우 약함. 본 단어는 통째 외워서 읽었던 듯. **글자→소리 매칭 자동화 안 되어 있음**.
- **자음 글자·소리 혼동 (신규 약점, 2026-05-04)**:
  - **b/d 혼동** — bat에서 첫 글자가 b인지 d인지 헷갈림
  - **f/p 혼동** — pan에서 f인지 p인지 헷갈림
  - 단모음보다 더 시급한 영역일 수 있음. 짧은 미니 게임 트랙 신설 필요.
- **단어 흡수 매우 느림 (2026-05-04 재확인)**: 단모음 a 단어 11개(cat·hat·bat·sat·pan·mat·bag·map·fan·cap·jam) 중 정착된 단어 거의 없음. **신규 단어 추가 보류, 같은 단어로 2~3세션 더 정착 우선**.
- **사이트워드 약점**: "Is this a map?"에서 is·this 발음 모름. the 외에도 is/this도 별도 트랙 필요.
- **응용력**: 패턴 들어도 새 단어에 적용 어려움. 음운 인식 자체 약함.
- **문법**: is/are, have/has 부정확
- **말하기·회화 (2026-05-04 부모 평가 긍정)**: "What's this? / It's a ___" 패턴 도움 됨. **더 늘려도 좋겠음** — "어떤 때 이런 표현 쓰는지" 맥락형, 쉬운 초등 수준 회화로.
- **듣기**: 쉬운 표현 부분 이해, input 노출량 부족
- **쓰기**: 소문자 알파벳 학습 중, 철자·글자 형태 혼동 (특히 b/d/p/q)
- **Dr. Seuss류**: 반복으로 외워서 읽은 것으로 재해석. 자력 디코딩 단계 아님.

### 지도 환경
- 부모가 영어권 거주 경험 → 발음·프리토킹 지도 가능
- 단, **부모 자료 자체에 전문용어(vowel team, CVC, IPA, 음가, blending 등)가 들어가면 부담** — 자료는 한국어로 풀어 쓰고, 발음 시범은 한국어 음 비유로

### 시스템 진행 방향 (D22, 2026-05-07 갱신)
- **신규 단어는 매 세션 2~3개로 점진 확장** (이전 5개 → 축소). 단어량은 영어 실력 핵심이므로 늘려가는 방향 유지.
- **누적 회전 강화** — 정착 안 된 단어는 매 세션 워밍업·문장·회화·게임에 등장. 정착된 단어는 가끔 점검.
- **자음 혼동 미니 트랙 신설** — b/d, f/p, m/n 등. 매 세션 2~3분 짧은 게임형.
- **사이트워드 카드 트랙 신설** — is, this, the, a, I, see, on, with 같은 고빈도. 매 세션 3~5장 누적.
- **맥락형 회화 확대** — 인사·요청·놀이·식사 등 상황별 표현. 1세션당 1상황. "어떤 때 쓰는 말인지" 한국어 안내.
- 이중모음(ai/ea/oo 등)은 **단모음·자음·사이트워드 안정 후 진입**
- 4기능 균형 유지하되 듣기·읽기에 살짝 더 비중 (input 우선)
- 한 세션에 새 패턴 1개만, 같은 단어를 여러 맥락에서 반복 노출 (응용력 훈련)

### 취약 포인트 (2026-05-07 재정렬)
