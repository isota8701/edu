## 1. 사고력 수학

### 1.1 학원별 공통 단원 시퀀스 (초2)

대표 학원(CMS/시매쓰/와이즈만/소마/필즈/영재사관)은 자체 교재를 사용하나, 대부분 **"수와 연산 → 도형 → 측정 → 규칙·패턴 → 확률·논리 → 문제해결 종합"** 의 순환 구조를 따릅니다. 한 학기에 6~8개 단원, 한 단원당 2~3주 분량입니다.

초2 1학기 표준 시퀀스 (일반화):

1. **수의 구조 심화** — 세 자리 수, 자릿값, 수직선, 수 어림
2. **연산 사고력** — 두 자리 덧셈·뺄셈을 모형(수모형, 가르기/모으기, 부분-전체)으로 재해석
3. **곱셈의 의미** — 묶어 세기, 배의 개념, 직사각형 배열 모델
4. **평면도형의 성질** — 변·꼭짓점, 직각, 다각형 분류, 칠교판/펜토미노 응용
5. **규칙 찾기** — 수열, 도형 패턴, 표·그래프 규칙, 주기성
6. **측정 입문** — 길이(cm/m), 시각과 시간, 단위 변환
7. **논리·분류** — 베다이어그램 단순형, 참/거짓, 분류 기준 만들기
8. **문제해결 종합** — 문장제, 거꾸로 풀기, 그림 그려 풀기

초2 2학기 표준 시퀀스 (일반화):

1. **네 자리 수와 자릿값 확장**
2. **곱셈구구 의미와 활용** (단순 암기보다 "왜 그 값이 되는지")
3. **분수의 도입** (전체-부분, 등분, 이름 붙이기까지)
4. **시간·달력**
5. **표·그래프 만들기와 읽기**
6. **공간 감각** — 쌓기나무, 위/앞/옆에서 본 모양, 거울/대칭
7. **퍼즐형 사고** — 마방진, 복면산, 길찾기, 수 배치 퍼즐
8. **종합 문제해결** — 비형식적 방정식(빈칸 □), 표 만들어 풀기, 패턴 일반화

### 1.2 사고력 문제 유형 카탈로그

학원 사고력 교재(CMS·시매쓰·소마 류)에서 반복적으로 등장하는 **유형 12종**(추상화):

| # | 유형 | 예시 구조 (일반화) | 학습 목표 |
|---|---|---|---|
| 1 | 수 배열·자릿값 퍼즐 | "1~9 카드로 가장 큰 세 자리 수 만들기" 변형 | 자릿값 직관, 경우 수 |
| 2 | 가르기·모으기 패턴 | 수 분해의 다양한 방법 찾기 | 연산 유연성 |
| 3 | 곱셈 모델 추론 | 사각형 배열, 묶음 그림에서 곱셈식 만들기 | 곱셈 의미 |
| 4 | 도형 자르고 붙이기 | 칠교, 펜토미노로 모양 채우기 | 공간 추론 |
| 5 | 도형 개수 세기 | 그림 안 삼각형/사각형 개수 (중복 포함) | 체계적 세기 |
| 6 | 규칙 찾아 다음 항 | 수열·도형열의 다음 항 예측 | 패턴 일반화 |
| 7 | 표·달력·시각 추론 | "오전 9시 출발, 1시간 30분 후 도착" | 시간 감각 |
| 8 | 문장제 (문제해결) | 그림 그려 풀기, 표 만들어 풀기 | 표상 변환 |
| 9 | 거꾸로 풀기 | "처음에 □개 → … → 마지막 ○개" | 역연산 |
| 10 | 분류·논리 | 두 가지 기준 동시 분류, 벤다이어그램 | 다중 기준 사고 |
| 11 | 마방진·복면산 | 합이 같도록 수 배치, 글자=숫자 | 제약 만족 |
| 12 | 길찾기·경우의 수 | 격자 위 최단경로 가짓수 | 조합 직관 |

학원별 색깔(일반화):

- **CMS·시매쓰**: 단원 한 권을 한 주제로 깊이 파고듦. **모델링·교구**(수모형·칠교·쌓기나무) 비중이 높음.
- **소마**: **퍼즐·게임형 비중**이 높고, 문제 표면이 친숙하나 함정이 많음.
- **와이즈만**: **STEAM·과학 융합**, 도형/측정/실생활 자료 해석 풍부.
- **필즈·황소**: **경시 대비형**, 난이도 곡선이 가팔라 상위권 타겟.
- **영재사관**: **영재고/특목 트랙 의식**, 반복 훈련 + 변별 문제 혼합.

(이 색깔 분류는 학부모 인식·블로그 통념 기반의 일반화이며, 실제 강의안과 다를 수 있음)

### 1.3 학교 진도와의 매핑

2022 개정 교육과정 기준 초2 수학 단원과 사고력 수학의 매핑(일반화):

| 학교 단원 | 학교에서 다루는 것 | 사고력 학원에서 추가로 다루는 것 |
|---|---|---|
| 세 자리 수 | 자릿값, 크기 비교 | 수 어림, 자릿값 퍼즐, 큰/작은 수 만들기 |
| 여러 가지 도형 | 변·꼭짓점, 직각 | 도형 분해·합성, 칠교, 펜토미노, 도형 개수 세기 |
| 덧셈과 뺄셈 | 두 자리 받아올림/내림 | 가르기·모으기 다양한 전략, 거꾸로 풀기, □ 빈칸 |
| 길이 재기 | cm·m | 단위 변환, 어림, 측정 도구 비교 |
| 분류하기 | 한 기준 분류 | 두 기준 분류, 벤다이어그램, 분류 기준 만들기 |
| 곱셈 | 묶어 세기 → 곱셈식 | 직사각형 배열 모델, 곱셈의 다양한 표상 |
| 네 자리 수 | 만 단위까지 | 자릿값·수 어림 심화 |
| 곱셈구구 | 2~9단 암기 | 곱셈구구 패턴(짝수단/5단), 곱셈표 빈칸 추론 |
| 길이 재기(2학기) | mm, km 도입(교과서마다) | 단위 환산, 측정 오차 |
| 시각과 시간 | 분 단위, 달력 | 시간 계산(앞·뒤), 시간 차 문제 |
| 표와 그래프 | ㅇ표/그림그래프 | 자료 정리·해석, 데이터로 결론 내기 |
| 규칙 찾기 | 단순 규칙 | 수열/도형열, 다중 규칙, 일반화 |

핵심 원리: 학원은 **학교 단원을 같은 시기 또는 1~2개월 선행**으로 다루되, **표상·전략·일반화** 차원을 더한다. 단순 풀이가 아니라 "왜·어떻게·또 다른 방법"을 묻는 형식.

### 1.4 권장 난이도 곡선

사고력 수학 한 단원(2주 분량)의 난이도 곡선(일반화):

```
난이도
  ▲
  │                                    ★ 도전(1~2)
  │                          ◆ 변별(2~3)
  │                ● 응용(3~4)
  │      ▲ 표준(5~6)
  │  ■ 도입·개념(2~3)
  └─────────────────────────────────▶ 차시
   1차시      2~3차시      4~5차시   6차시   7차시
```

문제 비중(권장):

- 도입·개념 이해: 15~20%
- 표준 적용: 35~40%
- 응용·전이: 25~30%
- 변별·심화: 10~15%
- 도전·경시형: 5~10%

원칙:

- **개념-적용-응용-심화** 4단 구조를 한 차시 안에서도, 단원 전체에서도 반복.
- 한 차시당 핵심 도전 문제는 **2문제 이내**로 제한하고, 나머지는 자신감 유지용.
- 같은 개념을 **다른 표상**(수식·그림·표·실물)으로 최소 3회 반복.

### 1.5 일반 연산 드릴과의 차별점

| 항목 | 연산 드릴 (구몬·기탄·눈높이) | 사고력 수학 |
|---|---|---|
| 목적 | 속도·정확도 | 표상 전환·전략 다양화 |
| 문제 형태 | 식 위주, 같은 패턴 반복 | 문장·그림·표·퍼즐 다양 |
| 답 | 유일 | 풀이가 다양 (답이 같아도 경로가 여러 개) |
| 시간 | 분당 답 수 측정 | "한 문제 10분 고민" 허용 |
| 평가 | 정답률·속도 | 풀이 과정·설명력 |

홈스쿨링 시 중요한 점: **연산 드릴(매일 10~15분) + 사고력(주 2~3회 30~60분)** 을 분리해 운영하는 것이 학원 모델의 핵심. 둘을 한 세션에 섞으면 어느 쪽도 제대로 안 됨.

## 2. 국어 독해·논술

### 2.1 지문 주제 풀

한우리·솔루니·시매쓰 국어 등에서 초2가 다루는 지문은 대체로 다음 풀에서 선택됩니다(일반화):

**문학 지문 (전체의 50~60%)**

- 동화 (전래·창작): 흥부와 놀부 변주, 우정·가족·용기 주제
- 동시: 자연, 계절, 학교생활, 가족
- 짧은 소설/생활문: 친구와의 갈등, 약속, 실수
- 위인전 단편: 세종대왕, 이순신, 헬렌 켈러, 에디슨 (초2 수준 축약)
- 우화·이솝: 교훈 명확, 토론 도입용

**비문학 지문 (40~50%)**

- 자연·과학: 동물 생태(개미·꿀벌·나비 변태), 날씨, 식물의 한살이, 우주 입문
- 사회·문화: 명절, 직업 소개, 우리 동네, 교통수단의 변화
- 인물·역사: 한 인물의 어린 시절 일화 중심
- 안내문·설명문: 도서관 이용법, 박물관 안내, 재활용 분리수거
- 실용문: 편지, 일기, 안내장, 광고문 (장르 인식 학습)
- 시사 입문: 환경(플라스틱·기후), 안전(학교폭력·교통), 공동체 (난도 조절된 버전)

**지문 길이 (초2 일반)**

- 1학기: 250~400자
- 2학기: 350~600자
- 단원 종합 회차에서는 700~900자 1편 + 짧은 자료 1~2개 결합형

### 2.2 문제 유형 단계 (사실 → 추론 → 토론)

학원 독해 워크시트는 보통 **3~5단 구조**를 따릅니다(일반화):

**단계 1: 사실 확인 (Literal)**

- 누가/언제/어디서/무엇을 했는지
- 글에 직접 나온 단어·문장 찾기
- 순서대로 사건 나열

**단계 2: 어휘 (Vocabulary)**

- 문맥에서 단어 뜻 추론
- 비슷한 말·반대말
- 짧은 문장에 단어 적용
- 다의어·동음이의어 입문

**단계 3: 추론 (Inferential)**

- 인물의 마음/생각/이유 추측
- 글 전체의 중심 생각
- 빠진 정보 채워 넣기
- 글쓴이의 의도

**단계 4: 적용·비판 (Critical)**

- "내가 ○○라면 어떻게 했을까"
- 두 인물·두 글 비교
- 글의 일부를 바꿔 다시 쓰기
- 사실과 의견 구분 (초2 후반부 도입)

**단계 5: 토론·표현 (Discussion/Production)**

- 짝·모둠 토론: "○○은 옳을까?" 형 질문
- 자기 생각 3~5문장 쓰기
- 인물에게 편지 쓰기
- 짧은 주장글 (주장 + 이유 1~2개)

토론형 진행 패턴(한우리·솔루니 류, 일반화):

1. **읽기 전(Pre)**: 표지·제목 보고 예측, 배경지식 끌어오기 (5분)
2. **읽기 중(While)**: 소리 내어 읽기 + 모르는 단어 표시, 짧은 멈춤-질문 (15~20분)
3. **읽기 후 1차(Comprehension)**: 사실+어휘 워크시트 (10~15분)
4. **읽기 후 2차(Discussion)**: 추론·열린 질문 모둠 토론 (15~20분)
5. **표현(Output)**: 짧은 글쓰기 또는 발표 (15~20분)
6. **공유·피드백**: 두 명 정도 낭독, 교사 코멘트 (5~10분)

총 80~100분 1회 수업.

### 2.3 초2 권장 지문 길이·어휘 수준

| 항목 | 1학기 | 2학기 |
|---|---|---|
| 지문 길이 | 250~400자 | 350~600자 |
| 한 차시 누적 분량 | 600~900자 | 900~1500자 |
| 어휘 난도 | 1~2학년 교과서 수준 + 알파 10~20% | 2~3학년 수준 혼합 |
| 한자어 노출 | 일상 한자어 위주(생활·자연) | 추상어 입문(까닭, 의견, 주장, 사실) |
| 신출 어휘/회차 | 5~8개 | 8~12개 |
| 문장 길이 | 평균 15~25자, 한 문단 3~4문장 | 평균 20~35자, 한 문단 4~6문장 |
| 글쓰기 분량 | 3~5문장 | 5~10문장 (짧은 한 문단) |

**작문 루브릭(일반화, 초2 후반)**

- 주제(중심 생각)가 분명한가
- 이유·근거가 1개 이상인가
- 처음-가운데-끝 구조가 있는가
- 띄어쓰기·맞춤법 (받아쓰기 수준 + 자주 틀리는 어미)
- 자기 경험·생각이 들어 있는가

### 2.4 어휘·받아쓰기 보조 트랙

대부분의 국어 학원은 **본 수업 외 받아쓰기·어휘 숙제**를 병행:

- 매주 받아쓰기 10~20문장 (교과서 + 학원 자체 단어장)
- 한자 1일 1자(8급~7급, 山·水·一~十 등)
- 속담·관용어 입문 (주 1~2개)
- 띄어쓰기 게임형 워크시트

## 3. 영어 파닉스·단어

### 3.1 파닉스 단계 시퀀스

대형 어학원(청담·폴리·SLP)은 자체 레벨 시스템을 쓰지만 파닉스 시퀀스는 거의 표준적입니다(일반화):

**Phase 1 — 알파벳·문자 인지 (Pre-Phonics)**

- 대문자/소문자, 글자 이름, 글자 소리(letter sound)
- 글자-소리 1:1 매칭 (a=/æ/, b=/b/ ...)

**Phase 2 — 단모음 CVC**

- a / e / i / o / u 단모음
- cat, pen, sit, hop, bug 패턴
- 단모음 + 자음 블렌딩 (b-a-t → bat)

**Phase 3 — 자음 블렌드·디그래프**

- 초성 블렌드: bl, cl, fl, gl, pl, sl, br, cr, dr, fr, gr, pr, tr, sk, sl, sm, sn, sp, st, sw
- 종성 블렌드: -nd, -nk, -nt, -mp, -st, -ft, -lt
- 디그래프: ch, sh, th, wh, ph, ng, ck

**Phase 4 — 장모음·매직 e**

- a_e, i_e, o_e, u_e (cake, bike, home, cute)
- 단모음 ↔ 장모음 비교 (cap/cape, hop/hope)

**Phase 5 — 모음팀(이중모음)**

- ai/ay, ee/ea, oa/ow, oo, ou/ow, oi/oy, au/aw
- "두 모음이 걸어가면 첫째가 말한다" 같은 mnemonic

**Phase 6 — R-controlled & 기타**

- ar, er, ir, or, ur
- y as vowel (happy, sky)
- 예외 단어 (sight words 일부)

**Phase 7 — 음절·접사**

- 1~2음절 끊어 읽기, 자석 음절(cap-tain)
- -s, -ed, -ing 어미
- 간단한 접두사 (un-, re-)

**초2 일반 진도 (일반화)**

- 영어 시작 1년차(초1 후반~초2): Phase 2~4
- 영어 시작 2년차(초2 후반~초3): Phase 5~7 + 챕터북 입문
- 유아기부터 시작한 어학원 상위반: 초2에 이미 챕터북·논픽션 리딩 단계

대치·잠실·목동의 대형 어학원 상위 트랙은 초2에 이미 **Magic Tree House, Junie B. Jones, Nate the Great** 류 챕터북을 읽는 경우가 많고, 일반 트랙은 **파닉스 마무리 ~ 리더스북(Step into Reading 1~2단계)** 수준이 평균(일반화).

### 3.2 단어 확장 트랙

학원에서 단어 확장은 보통 **3축 병행**:

**축 A: 사이트워드 (Sight Words)**

- Dolch / Fry 빈도 리스트 기반
- Pre-K, K, Grade 1, Grade 2 단계
- 초2 목표: Grade 1 완전 + Grade 2 진입 (대략 200~300 단어)

**축 B: 주제별·테마 어휘**

- 가족·집·학교·동물·음식·날씨·계절·직업·교통·신체·옷·감정
- 한 테마당 10~20단어, 그림 카드 + 짧은 문장 적용
- 초2: 10~15개 테마 / 학기

**축 C: 챕터북·리더스북 어휘**

- 읽는 책에서 신출 단어 추출 → 단어장
- 챕터북 1권당 30~80 신출 단어
- 정의 + 예문 + 자기 문장 만들기

**초2 목표 어휘 누적량 (일반화)**

| 트랙 | 1년차 (영어 입문) | 2~3년차 (어학원 정규) | 5년차+ (선행반) |
|---|---|---|---|
| 누적 어휘 | 200~400 | 600~1000 | 1500~2500 |
| 사이트워드 | Pre-K~K | Grade 1~2 | Grade 2~3 완료 |
| 챕터북 | 미진입 | I Can Read 2~3, ORT 5~8 | Magic Tree House 등 |

### 3.3 듣기·말하기·읽기·쓰기 비중

학원 유형별 4기능 비중(일반화):

| 학원 타입 | Listening | Speaking | Reading | Writing |
|---|---|---|---|---|
| 대형 어학원 (청담·폴리·SLP) | 25% | 25% | 30% | 20% |
| 학습지 기반 (윤선생·이튼) | 30% | 20% | 35% | 15% |
| 동네 공부방·소형 | 20% | 15% | 45% | 20% |
| 주니어 영어유치원 출신 트랙 | 30% | 35% | 20% | 15% |

**초2 권장 비중 (홈스쿨링 적용)**

- 듣기: 25~30% (영상·오디오북 인풋, 매일 15~20분)
- 말하기: 15~20% (소리 내어 읽기·쉐도잉·간단 문답)
- 읽기: 30~35% (파닉스 워크북 + 리더스북)

쓰기는 초2에서 **과욕 금물**. 한 문장 정확히 쓰기 → 두 문장 잇기 → 5문장 글까지 1년 단위로 천천히.

## 4. 보조: 사고력 과학·인문학

### 4.1 초2 주제 풀

대표 학원: 주니어플라톤(인문·논술), 와이즈만(과학·STEAM), 창의과학·CSI 류.

- 생물: 곤충(개미·꿀벌·나비), 식물의 한살이, 우리 몸(뼈·근육·소화 입문), 동물 분류
- 물질: 고체·액체·기체, 물의 변화, 자석, 빛과 그림자, 소리
- 지구·우주: 날씨와 계절, 달의 모양, 태양계 입문, 흙과 돌
- 환경·생활: 분리수거, 에너지 절약, 바다 오염, 생물 다양성
- 기술·일상: 수레바퀴·도르래(간단 도구), 자동차의 원리, 소화기 등 안전

**인문학·사회 (초2 수준)**

- 자기 이해: 나의 감정, 우정, 가족, 약속
- 공동체: 우리 동네, 교통, 직업, 다문화
- 윤리: 거짓말·정직, 공정과 차례, 약자 배려
- 역사 입문: 우리나라의 옛날 이야기, 시대별 의식주 변화, 위인 일화
- 철학 입문(주니어플라톤 류): "친구는 왜 필요할까", "약속은 꼭 지켜야 할까", "좋은 거짓말이 있을까"
- 경제 입문: 용돈, 저축, 시장과 가게, 직업과 일

### 4.2 활동 형식

학원 1회 수업(80~120분) 표준 흐름(일반화):

**과학 수업**

1. 도입 — 영상 1~2분, 질문 던지기 (5분)
2. 관찰·탐색 — 실물·사진·영상 자료, 자유 관찰 (10~15분)
3. 핵심 개념 설명 — 짧고 명확히 (10분)
4. 실험·활동 — 손으로 만져 보는 단순 실험 (30~40분)
5. 기록 — 관찰일지, 그림 + 한 줄 설명 (15분)
6. 공유 — 한두 명 발표, 질문 (10분)
7. 마무리 — 오늘 배운 것 한 문장 (5분)

**인문학·논술 수업**

1. 도입 — 짧은 동화·이야기 또는 사진 자료 (5~10분)
2. 함께 읽기 — 낭독 또는 교사 낭독 (10분)
3. 1차 질문 — 사실 확인 (5~10분)
4. 핵심 토론 질문 — 1~2개, 모둠 토론 (20~30분)
5. 자기 생각 정리 — 짧은 글쓰기 (15~20분)
6. 발표·반박·재고 (10~15분)
7. 마무리 — "오늘 가장 인상 깊었던 한마디" (5분)

**활동 형식 카탈로그 (재사용 가능한 9종)**

1. 관찰 → 그리기 → 한 줄 설명
2. 실험 → 가설/결과 비교표
3. 마인드맵 (한 단어에서 가지치기)
4. 비교표 (둘 이상 대상 비교)
5. 입장 바꿔 쓰기 (인물·사물의 1인칭)
6. 토론(찬반) → 자기 입장 정리 글
7. 만들기·디자인 (재활용품으로 ○○ 만들기 류)
8. 문제 해결 시나리오 ("만약 ~라면")
9. 인터뷰 (가족·친구에게 질문하고 정리)

## 6. 콘텐츠 생성 시 적용 가이드 (핵심 요약)

> 모든 subagent(수학·국어·영어·과학·인문)가 빠르게 참고할 수 있는 핵심 원칙

1. **연산과 사고력은 분리하라.** 매일 10~15분 연산 드릴 + 주 2~3회 30~60분 사고력 세션이 학원 표준이다. 한 세션에 섞으면 안 된다.
2. **한 개념은 최소 3가지 표상으로.** 수식·그림·표·실물 중 3가지 이상으로 같은 개념을 반복한다(특히 수학·과학).
3. **난이도 곡선은 도입 20% / 표준 40% / 응용 25% / 심화 15%.** 도전 문제는 한 차시당 2개 이내로 제한해 학습 의욕을 보호한다.
4. **국어는 사실→어휘→추론→비판→토론 5단 구조.** 초2는 5단계 전체를 가볍게 다 거치되, 단계 4~5는 분량을 짧게(3~5문장 글쓰기).
5. **영어는 4기능 균형 + 인풋 우선.** 듣기 25~30% / 말하기 15~20% / 읽기 30~35% / 쓰기 15~20%. 쓰기는 욕심 금물.
6. **파닉스는 7단계 시퀀스.** 단모음 CVC → 블렌드/디그래프 → 매직 e → 이중모음 → R-controlled → 음절·접사 순으로, 학습자 시작 시점에 따라 진입점만 다르게.
7. **표절 금지, 패턴만 차용.** 문제 텍스트·도서 목록을 복제하지 말고 "유형 카탈로그(수학 12종, 국어 5단계, 활동 9종)"의 추상화된 구조만 참고한다.
8. **학교 진도와 1~2개월 선행 + 표상·전략 심화.** 학교 단원을 같은 시기에 다루되, "왜·또 다른 방법·일반화"를 더한다.
9. **세션 길이는 80~120분이 학원 표준.** 홈스쿨링은 집중력 한계로 20~40분 모듈을 2~3개 이어 붙이는 방식이 현실적.
10. **출력물은 항상 가시화한다.** 사고력 워크북, 독서기록장, 영어 단어장, 관찰일지처럼 **한 회 1산출물** 원칙. 진도 추적과 동기 부여에 필수.
